# linkerdog_ppt

Slidev deck for the 2026-06-22 LinkerDog midterm review:

- Slock-incubated meeting entry
- real-meeting demo target with golden fallback
- execution output plus memory reuse
- team memory as the north-star metric

## Usage

```bash
make install
make dev
```

Build static slides:

```bash
make build
```

Export PowerPoint:

```bash
make pptx
```
