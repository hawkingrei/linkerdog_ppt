# linkerdog_ppt

Slidev deck for a 3-minute Linkerdog pitch:

- bot for agent
- agent-native meeting context
- online meeting bot, historical video records, and IoT recording devices
- meeting minutes, suggestions, and durable memory

## Usage

```bash
make install
make dev
```

Build static slides:

```bash
make build
```

Export PowerPoint:

```bash
make pptx
```
